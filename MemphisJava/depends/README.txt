SETUP INSTRUCTIONS

In order to properly run the project, the depends folder must be added to the classpath.
(1) Right click MemphisCore.java
(2) Go to "Run As > Run Configurations..."
(3) Select the "Classpath" tab.
(4) Click "User Entries"
(5) Click the "Advanced" button on the right.
(6) Select the "Add Folders" radio button and click Ok.
(7) Browse to the "depends" folder and click Ok.
